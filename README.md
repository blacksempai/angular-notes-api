# Angular Notes
Simple Web-app for making a notes. 

## Technologies
Front-end:
* Angular 11
* TypeScript
* SCSS

Back-end:
* NodeJS
* TypeScript
* Express

## Run instructions
Clone repo:
```
git clone https://github.com/blacksempai/angular-notes-fullstack.git
```

Install dependencies:
```
npm i
```
```
npm run client-install
```

Run development server:
```
npm run dev
```
